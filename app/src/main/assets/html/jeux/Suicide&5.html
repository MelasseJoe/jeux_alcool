Les règles du jeu

Vous allez voir que ce jeu porte bien son nom.

Petit conseil : si vous n'avez pas de dés, vous pouvez télécharger notre générateur de nombre aléatoire pour remplacer les dés.

Etape 1 : Faites un lancer de dés, le nombre indiqué par le ou les dés indique le nombre de gorgées à boire.

Etape 2 : Faites un nouveau lancer de dés, le nombre indiqué par le ou les dés indique le nombre de pompes à faire. Recommencez maintenant à l'étape 1.
