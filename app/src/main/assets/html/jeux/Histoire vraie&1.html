Les règles du jeu

À tour de rôle, tous les joueurs vont raconter une histoire. Les autres vont devoir essayer de deviner si cette histoire est vraie ou fausse.
À chaque fin d'histoire tous les joueurs qui se trompent boivent une gorgée.

Si personne ne se trompe le conteur boit 1 gorgée.
