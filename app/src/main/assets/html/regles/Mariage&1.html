Le Mariage

La règle

Pour cette règle vous devez être un nombre pair.
Chaque joueur doit se marier avec un autre. Lorsque l'un des deux époux devra boire, le nombre de gorgées sera divisé entre les deux époux.
Si un nombre impaire de gorgée est donné, à eux de se mettre d'accord pour savoir qui boira le plus.
